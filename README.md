# SEO-Navigator : персональный AI-эксперт для роста бизнеса без лишних затрат

**Идея проекта:**
SEO-Navigator — это интеллектуальный веб-сервис на базе AI-агента, предназначенный для автоматизации базового поискового аудита сайтов. Он заменяет первичную консультацию и аудит дорогого SEO-специалиста. Система работает просто: пользователь вводит адрес сайта, а ИИ-агент анализирует его и составляет конкретный список улучшений. Сервис переводит сложный язык терминов на простой для понимания и дает инструменты для самостоятельного внедрения правок.

**Целевая аудитория:**

Владельцы малого бизнеса — кофейни, мастерские, клиники и любой другой бизнес, у которых есть сайт, но нет бюджета на штатного сотрудника/агентское сопровождение.

Предриниматели — те, кто запускает свои проекты в одиночку и вынужден выполнять роль и разработчика, и маркетолога одновременно.

Маркетологи/контент-менеджеры — сотрудники без необходимых знаний для самостоятельной настройки.

### Ключевые экраны и функциональность

| Экран | Что можно делать |
| :--- | :--- |
| **Экран запуска анализа** | Ввести URL-адрес своего сайта, запустить процесс сканирования ИИ-агентом и ознакомиться с примерами работы сервиса. |
| **Витрина рекомендаций** | Просматривать список найденных точек роста, фильтровать задачи по категориям (Техчасть, Контент, Ссылки) и добавлять их в свой план. |
| **Карточка задачи** | Изучать, почему эта правка важна для бизнеса, и получать пошаговую инструкцию по самостоятельному внедрению. |
| **План действий** | Работать с отобранным списком задач как с чек-листом, отмечать выполненное и видеть общий прогресс оптимизации сайта. |

## Пользовательские сценарии

Как владелец магазина, я хочу увидеть список технических ошибок моего сайта, чтобы понять, почему он медленно грузится и теряет клиентов.

Как маркетолог , я хочу видеть все рекомендации, чтобы быстро сделать те правки, которые не требуют навыков программирования.

Как предприниматель, я хочу добавить конкретные задачи по контенту в «План действий», чтобы видеть итоговый объем работ и контролировать их выполнение.
