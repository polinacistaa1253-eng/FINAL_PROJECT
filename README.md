# SEO-Navigator : персональный AI-эксперт для роста бизнеса без лишних затрат
**Идея проекта:**
SEO-Navigator — это интеллектуальный веб-сервис на базе AI-агента, предназначенный для автоматизации базового поискового аудита сайтов. Он заменяет первичную консультацию и аудит дорогого SEO-специалиста. Система работает просто: пользователь вводит адрес сайта, а ИИ-агент анализирует его и составляет конкретный список улучшений.
**Целевая аудитория:**
Владельцы малого бизнеса — кофейни, мастерские, клиники и любой другой бизнес, у которых есть сайт, но нет бюджета на штатного сотрудника/агентское сопровождение.
Предриниматели — те, кто запускает свои проекты в одиночку и вынужден выполнять роль и разработчика, и маркетолога одновременно.
Маркетологи/контент-менеджеры — сотрудники без необходимых знаний для самостоятельной настройки.
